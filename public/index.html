Toys API

All Optional Routes




# HTTP Method Route Description
1 GET /toys Gives you a list about the all toys in the collection
2 GET /toys/search Returns a list of games matching to the search query
3 GET /toys/category Returns a list of games by category
4 GET /toys/price Returns a list of games filtered by price range.
5 GET /toys/single/:id Returns a single game by its ID.
6 GET /toys/count Returns the number of toys
7 POST /toys Adds a new game.
Token is required!
8 PUT /toys/:id Updates an existing game.
Token is required!
9 DELETE /toys/:id Deletes a toy by its ID.
Token is required!



Users API

All Optional Routes




# HTTP Method Route Description
1 GET /users Test endpoint to check if the server is running
2 GET /users/userInfo Get the information for the authenticated user (requires authentication)
3 POST /users Create a new user
4 POST /users/login Authenticate a user and receive a JWT token in response
